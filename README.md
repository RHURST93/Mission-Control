
# 🛰️Drone Detection Data Fusion Platform

“This system is designed as a real-time data fusion pipeline.
Sensor data is generated by a simulator and streamed through Kafka. The backend, built with FastAPI, consumes that data, computes relationships, clusters using DBSCAN, and anomaly detection, and exposes everything over a WebSocket stream.
On the frontend, I built a React-based operator dashboard using Mapbox for geospatial visualization and a force-directed graph to show relationships between entities.
The UI includes features like swarm detection, prediction paths, anomaly alerts, and timeline replay.
Everything runs in containers and is orchestrated with Kubernetes so the system is modular and scalable.

## Features
- WebSocket real-time streaming
- Mapbox 3D visualization
- Swarm detection (clusters → polygons)
- Graph relationships overlay
- Prediction cones visualization
- Time playback system
- Alert system (severity-based)
- Multi-panel UI
- Kafka-ready backend
- Kubernetes deployment
